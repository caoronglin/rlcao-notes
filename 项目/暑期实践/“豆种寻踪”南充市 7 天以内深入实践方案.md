## 一、项目概述
### 1. 项目名称
**豆种寻踪·南充：川北丘陵大豆种质资源、豆玉复合种植与红色精神实践调研**
### 2. 实践区域
以南充市为主，不跨市扩展。重点区县如下：

| 区县    | 功能定位           | 核心任务                   |
| ----- | -------------- | ---------------------- |
| 顺庆区   | 市级对接、科研访谈、红色导入 | 南充市农业科学院/大豆研究所、罗瑞卿纪念馆  |
| 嘉陵区   | 豆玉复合种植现场学习     | 豆玉带状复合种植培训/示范点、田间观察    |
| 南部县   | 大豆生产主场         | 大豆/豆玉复合种植示范田、农户和合作社访谈  |
| 西充县   | 品种与制种线索        | 南夏豆25制种/品种展示/宜机化改造线索核查 |
| 仪陇县   | 红色资源主场         | 朱德故里、张思德纪念馆/故居、红色精神访谈  |
| 嘉陵江沿线 | 野生大豆线索核查       | 文献、标本、地方志、专家访谈和有限度生境观察 |

### 3. 核心判断
南充适合做“**栽培大豆 + 品种推广 + 田间调研 + 红色资源 + 野生大豆线索核查**”的融合方案。  
>但目前未核到南充市或各区县公开确认的“野生大豆分布点/标本号/保护点”。因此，野生大豆部分必须表述为**线索核查**，不能写成“已确认发现”或“计划采集”。

## 三、总体研究问题
### 1. 大豆生产与技术推广
1. 南充大豆生产主要落在哪些县区和种植模式？
2. 南夏豆25在南充及周边推广中体现出哪些品种优势？
3. 豆玉带状复合种植在川北丘陵区的主要技术难点是什么？
4. 农户是否愿意继续种植大豆？影响因素是收益、补贴、机械化，还是市场销售？
5. 市农科院、县农业农村局、合作社和农户之间如何形成技术推广链条？
### 2. 种质资源与野生大豆线索
1. 南充市农业科学院如何保存、评价和利用大豆种质材料？
2. 南夏豆25的亲本来源、性状优势和适宜区域是什么？
3. 南充是否存在可核实的野生大豆公开记录、标本记录、文献记录或地方口述线索？
4. 嘉陵江流域是否存在适合开展豆科植物生境观察的河岸、湿地、田边、灌丛等环境？
5. 在没有确证记录的情况下，怎样规范表述“野生大豆线索核查”？
### 3. 红色资源与大思政课
1. 朱德精神、张思德精神、罗瑞卿精神如何与新时代粮食安全结合？
2. 农业科技推广是否可以理解为新时代“为人民服务”的实践场景？
3. 青年学生如何把作物学、种业、农技推广知识用于乡村振兴？
4. 红色资源如何避免停留在参观层面，转化为具体调研问题和实践成果？

## 四、7 天以内详细行程
建议采用“**6 天主方案 + 第 7 天机动**”。若学校要求完整 7 天，可执行 D1—D7；若经费或时间紧张，可压缩 D6。
### D1｜顺庆区：项目启动、专家访谈、红色导入
**上午：南充市农业科学院 / 大豆研究所**
目标：建立专业调研基础，确定后续区县样点。  
任务：
- 访谈南充市农业科学院大豆研究所专家；
- 了解南夏豆25选育背景、推广区域、适应性和技术要点；
- 咨询南部县、嘉陵区、西充县适合走访的田块、基地或合作社；
- 咨询南充是否有野生大豆、近缘资源、标本或地方记忆线索；
- 明确哪些材料可公开引用，哪些只作内部参考。
建议访谈问题：
1. 南夏豆25相较于普通大豆品种的主要优势是什么？
2. 川北丘陵区推广大豆最大的制约因素是什么？
3. 豆玉复合种植中，品种、密度、行比和机械化如何匹配？
4. 南充是否曾发现过野生大豆或疑似野生大豆？
5. 学生团队适合去哪些县区做田间观察？
**下午：罗瑞卿纪念馆（顺庆区）**
目标：完成红色资源导入。  
任务：
- 采集罗瑞卿纪念馆红色全景脚本素材；
- 记录“纪律、担当、国家安全、粮食安全”的思政连接点；
- 形成“红色精神为什么能进入粮食安全调研”的开篇逻辑。
当天成果：
- 专家访谈纪要 1 份；
- 南充大豆调研样点初选表 1 份；
- 红色资源导入素材 1 组。

### D2｜嘉陵区：豆玉带状复合种植现场学习
目标：理解豆玉复合种植如何从技术方案落到田间。
建议对接对象：嘉陵区农业农村局、农技推广站、示范田负责人、种植大户或合作社。
观察内容：

| 类别 | 记录要点 |
|------|----------|
| 种植模式 | 大豆与玉米行比、行距、密度、播期 |
| 品种 | 大豆品种、玉米品种、是否涉及南夏豆25 |
| 田间管理 | 施肥、除草、病虫害、防倒伏、灌排条件 |
| 机械化 | 播种机、植保机、收获机适配情况 |
| 技术服务 | 是否有现场培训、专家授课、微信群指导 |
| 农户反馈 | 收益、补贴、劳动量、技术难度、继续种植意愿 |
当天成果：
- 田间观察表 1 份；
- 农技人员访谈 1—2 人；
- 农户或合作社访谈 2—3 人；
- 豆玉复合种植照片/短视频素材 1 组。

### D7｜嘉陵江沿线：野生大豆线索核查与成果汇总
目标：规范完成“野生大豆”部分，不夸大、不采集、不预设结果。
核查内容：
1. **专家访谈**  
   向南充市农科院、四川省农科院、四川农业大学等专家确认：南充是否有公开野生大豆记录或疑似线索。
2. **文献核查** 
   检索关键词：南充、嘉陵江、野生大豆、豆科植物、标本、四川盆地、大豆近缘资源。
3. **标本核查**  
   询问高校标本馆、地方农业部门或自然资源相关部门是否有馆藏或历史记录。
4. **有限度生境观察** 
   在嘉陵江沿线观察河岸、湿地、田边、灌丛等潜在生境。仅做拍照、定位、环境记录，不采集、不传播敏感点位。
5. **结论表述**  
   - 若未发现证据：写“截至本次调研，未核到南充野生大豆公开确认记录。”
   - 若发现疑似线索：写“发现疑似生境或口述线索，需专家鉴定和主管部门确认。”
当天成果：
- 野生大豆线索核查表；
- 专家访谈纪要；
- 嘉陵江沿线生境照片；
- “不采集、不扰动、不公开敏感点位”承诺记录；
- 全队复盘和成果分工表。

### 4. 路线优化建议
若高德核验后发现车程偏长，按以下方式压缩：
1. **D3 南部县当天往返困难：** 当晚住南部县，D4 从南部转西充，再回南充主城区。
2. **D5 仪陇点位过满：** 只保留朱德故里 + 张思德纪念馆，张思德故居放入 D6 机动。
3. **D6 阆中车程过长：** 取消阆中，改为仪陇补访或主城区整理。
4. **D7 返程压力大：** 嘉陵江生境观察压缩为 1—2 小时，只保留公共安全点位。

### 4. 对 7 天行程的实际影响
1. **D1 更稳：** 农科院到罗瑞卿纪念馆约 15 分钟，D1 可同时完成专业对接和市区红色资源记录。
2. **D2 不需要担心主城区到嘉陵区车程：** 浏览器路线显示约 20 分钟级别，真正风险是示范田/合作社是否有明确接待点。
3. **D3 南部县需早出发：** 主城区到南部县城约 75—78 km、1 小时 16—21 分钟；若再进田块，当天任务会偏满。
4. **D4 西充可以承接 D3：** 南部到西充约 1 小时左右，适合做“南部—西充”连续农业调研线，但村点必须由县区农业农村局或农科院推荐。
5. **D5-D6 仪陇红色线不能压得太紧：** 南充到朱德故里约 1.5 小时，朱德故里到张思德纪念馆约 1 小时，两个点建议分主次。
6. **阆中仍是可选项：** 南充到阆中古城约 100 km、1 小时 18 分钟左右，不应挤占朱德故里/张思德线索核查时间。
